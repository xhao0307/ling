# 童话儿童绘本风 Prompt 指导手册（CityLing）

本手册用于统一所有插画生成任务的视觉基调，避免“同质化角色”“出现文字水印”“风格漂移”。

## 1. 风格定位（固定）
- 关键词: 儿童绘本、手绘、水彩+蜡笔、柔光、温暖、梦幻、治愈。
- 配色倾向: 低饱和粉/蓝/薄荷/奶油黄，避免高对比霓虹色。
- 画面气质: 童真、安全、鼓励探索，不做写实摄影感。

## 2. Prompt 结构（固定模板）
每次生成必须包含四段:
1. `base_prompt`（全局基调）
2. `style_anchor`（世界观一致性）
3. `negative_prompt`（禁用项）
4. `asset.prompt`（单图主题）

推荐骨架:

```text
base_prompt: 儿童绘本手绘风，水彩+蜡笔混合笔触，柔和粉蓝薄荷配色，温暖童话光感，轻留白。
style_anchor: 同一童话世界观与笔触，但每张图主角身份、动作、构图都不同。
negative_prompt: no text, no letters, no words, no watermark, no logo, no signature, no subtitle, 禁止任何文字与水印，禁止写实照片感，禁止重复同一小女孩主角。
asset.prompt: [页面语义] + [主角设定] + [动作] + [构图位置] + [留白要求]。
```

## 3. 主角多样化规则（强制）
- 同一批次不得反复使用同一人物原型（例如同发型小女孩）。
- 至少覆盖 3 类主角: 男孩/女孩、动物、幻想生物。
- 连续两张图不能同时重复:
  - 主角身份
  - 主动作
  - 主构图（如“角色居中站立”）

## 4. 页面类型构图基线
- `login`: 右侧或顶部留白，便于叠加按钮和输入框。
- `empty_state`: 主体更聚焦，情绪偏“等待/引导”。
- `onboarding`: 动线明确（从左到右或由近到远），突出“探索”。
- `achievement`: 图标化、中心构图、边缘干净，适合徽章裁切。
- `background`: 低干扰细节，避免喧宾夺主，方便叠加前景 UI。

## 5. 禁止项（硬约束）
- 任何文字、字母、手写体、Logo、水印、签名、右下角标记。
- 写实摄影质感、3D 游戏材质感、恐怖/压迫氛围。
- 与参考图完全同构图、同角色、同动作的“复刻”结果。

## 6. 质量自检（出图后）
- 风格一致性: 笔触和色调是否同属一套绘本体系。
- 主题差异性: 主角、动作、场景是否明显不同。
- 生产可用性: 是否具备文案留白、是否可直接进 UI。
- 污染检查: 是否出现文字、水印、签名痕迹。

未通过任一项时，必须重采样并加强 `negative_prompt`。
